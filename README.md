# KubeHound Demo

Demo environment for KubeHound security analysis using Kubernetes Goat.

## Quick Start

```bash
./setup-demo.sh
```

## What This Demo Does

- Creates a Kind cluster with Kubernetes Goat (vulnerable by design)
- Deploys KubeHound to analyze the cluster
- Generates attack path graphs
- Provides a complete security analysis demo environment

## Prerequisites

- Docker
- Kind
- Helm
- KubeHound
